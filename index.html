Things I Built

NextGen Health Companion is an AI-powered smart healthcare web application developed to help users monitor and manage their health more effectively. The system provides personalized diet plans, workout recommendations, food image analysis, and disease risk prediction using AI technologies. Built using HTML, CSS, JavaScript, Python Django, MySQL, and AI integration, the platform includes separate modules for users, dietitians, medical experts, and admins for smooth system management.
The project was designed to improve health awareness and make health tracking easier through a user-friendly digital platform. Users can securely store and manage their personal health information, analyze food images for nutritional details, and receive smart health recommendations based on their data. This project also helped improve practical skills in frontend development, backend development, database management, and AI-based application integration while working in a team environment.
